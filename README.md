# ManageRoom
ManageRoom help manage members, notify by email in the room
## Use the package manager (npm)[https://www.npmjs.com/] to install 
```bash
1. npm install
2. cd client
npm install 
```
## Get started
```bash
1. cd client 
  npm start
2. node server
```
